# Advance-Excel-Project
## Tools & Technologies
- **Microsoft Excel**: Main tool for data analysis and formula implementation.
- **Excel Functions**: VLOOKUP, XLOOKUP, INDEX, MATCH, IF, IFS, etc.

## Challenges Faced
- Handling #N/A errors in lookup functions.
- Managing large datasets with multiple criteria lookups.
- Ensuring compatibility across different Excel versions.

## Insights Gained
- XLOOKUP is more flexible and powerful than VLOOKUP for modern Excel versions.
- Combining lookup functions with logical and text functions enhances data analysis capabilities.
- Proper use of tables and named ranges improves formula readability and maintenance.
